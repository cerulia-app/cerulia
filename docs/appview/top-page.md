# トップページ設計

## `/` (public top)

匿名で見えるランディングページ。

### First viewport

1. **1 行の約束**: 「TRPG のキャラクターを作って、遊んだ記録を残して、見せよう」
2. **1 つの具体例**: 公開されているキャラクター、または明示ラベル付き synthetic sample の立ち絵とセッション履歴カード
3. **2 つの CTA**: 「始める」（sign-in）と「キャラクターを見る」（公開キャラクター detail へ）

### 状態ごとの扱い

- 公開キャラクターがない場合: 「サンプル表示」と明示した synthetic fixture を表示する
- sign-in 済みの場合: `/home` にリダイレクト

## `/home` (signed-in home)

sign-in 後の canonical landing。

### 構成

1. **継続作業**: 最近開いた branch や最近の session への再入導線
2. **最近のセッション**: 直近のセッション記録
3. **CTA**: 「キャラクターを作る」「セッションを記録する」
